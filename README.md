# SQLite Analytics Challenge

## Overview

This project uses SQL queries in SQLite to analyze customer spending, product revenue, employee salaries, and customer loyalty trends using the provided business database.

## Files Included

* `challenge.sql` – Contains SQL solutions for all four required tasks.
* `INSIGHTS.md` – Summary of business insights based on query results.
* `bais_sqlite_lab.db` – Provided SQLite database file.

## Tools Used

* SQLite
* SQLTools extension in VS Code

## Validation

All queries were tested successfully against the provided database and executed without errors.

## Tasks Completed

1. Top 5 customers by total lifetime spend
2. Total revenue by product category
3. Employees earning above department average
4. Cities with the most Gold loyalty customers
